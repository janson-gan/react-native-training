* **Control flow**
  * Using control flow to validate the defect.
    * First, need a method to check what happen when rgb color index go beyond 255, and the best candidate goes to 'if-else' statement! üëè
    * By using 'if-else' statement, we can perform a task when rgb color index go beyond 255.
    
     ```
     if (red + COLOR_INCREMENT > 255) {
      //do something here...
      return;
     }
     ```
     * In the above statement, we tell the 'validator', if the rgb color index hit 255 then do nothing further and stop there even if user keep clicking the increase button. 
     * And yes, its works! Rgb color red index stop at 255 despite increase button keep clicking, and decreasing the color index now is much more easier and is immediate effect. This method can apply to all buttons including decrease button, just have to replace the '+' to '-' operator. üëç
